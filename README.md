# 码道 MaDao — The Way of Code

> 从零到智能体 / From Zero to Agentic AI

A bilingual hub that arranges five Claude Code tutorials into a single learning path: **Foundations → Practice → Advanced**, with two flavors per stage.

| Stage | Choose one | |
|---|---|---|
| 1 · Foundations | [起码 QiMa](https://claude-code-launch.pages.dev/) | [AI 起 AiQi](https://aiqi.pages.dev/) |
| 2 · Practice | [码功 CodeFu](https://codefu.pages.dev/) | 码玩 CodePlay *(soon)* |
| 3 · Advanced | [智路 AgentPath](https://agentpath.pages.dev/) | |

Vanilla HTML + Tailwind v4 + ~250 lines of JS. No framework, no backend. Built with Claude Code.

```bash
npm install
npm run build:css
python3 -m http.server 8765
```

See `CLAUDE.md` for project conventions.
