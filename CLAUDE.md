# 码道 MaDao

Unified hub for the five Claude Code tutorials: 起码 QiMa · AI 起 AiQi · 码功 CodeFu · 码玩 CodePlay · 智路 AgentPath. Single static page that arranges them as a 3-stage path (Foundations → Practice → Advanced), with pick-one framing in stages 1 & 2.

## Tech Stack
- Vanilla JS (ES modules) + HTML, no framework, no TypeScript
- Tailwind CSS v4 via `@tailwindcss/cli` → `dist/tailwind.css`
- localStorage for language pref (`madao:lang`) and per-course "Done ✓" (`madao:done:<id>`)
- Playwright for build-time thumbnail capture (not runtime)
- Cloudflare Pages, project name `madao-cc` (madao.pages.dev was taken)

## Structure
```
MaDao/
├── index.html               # entry — header, hero, #stages mount, FAQ, footer
├── src/
│   ├── app.js               # init: i18n, render stages, attach handlers
│   ├── i18n.js              # bilingual strings + html[lang] swap
│   ├── content.js           # COURSES + STAGES — single source of truth
│   ├── render.js            # builds stage + card DOM from content.js
│   ├── progress.js          # localStorage Done toggle, class swaps
│   └── styles.css           # Tailwind v4 + Vibe B custom (cream/ink/coral)
├── public/thumbs/           # 16:9 PNGs (auto-generated by snap-thumbs)
├── scripts/
│   ├── snap-thumbs.mjs      # Playwright: 4 live screenshots + 1 placeholder
│   └── _dev-screenshot.mjs  # local QA script (not deployed; runs against :8765)
├── dist/tailwind.css        # built CSS (committed for CF Pages)
├── package.json             # build:css, dev:css, snap
└── wrangler.toml            # name = "madao-cc"
```

## Conventions
- **Bilingual zine**: hero + course names + stage labels show ZH and EN simultaneously. Body copy (FAQ, pitches, "pick if…", durations) swaps via `data-i18n` on toggle.
- **Toggle button** shows the CURRENT language ("中" in zh mode, "EN" in en mode), not the opposite — matches CLAUDE.md global rule.
- **html[lang]** updated in `initI18n()` and on every `setLanguage()` — needed for screen readers' voice engine selection.
- **Single source of truth**: `src/content.js` exports `COURSES` and `STAGES`. Adding/removing a course = edit content.js, no HTML touch.
- **Visual identity**: cream `#FAF6EE` paper, ink `#1A1A1A`, coral `#D9534F`. Box-shadow offset 6px for the zine "stamp" feel. Hand-drawn arrows are inline SVG with a turbulence filter.
- **No backend**, **no auth**, **no analytics** in v1.

## Build & Run
```bash
npm install
npm run build:css                     # one-time CSS build
python3 -m http.server 8765           # any static server
npm run snap                          # regenerate thumbnails (Playwright)
```

## Deployment
```bash
npm run build:css                     # ensure dist/tailwind.css is fresh
wrangler pages deploy . --project-name=madao-cc
```
After first push: `git remote set-head origin --auto`.

## Out of scope (v1)
- Cross-app progress, accounts, certificates, search, dark mode, analytics, recommender. The 3-stage structure already steers learners.
